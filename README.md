#### Initialisation de projet :
 `yarn init --yes`

#### Ajout Typescript et tslint

`yarn add --dev typescript tslint` <br/>
`yarn add express` <br />
`yarn add --dev @types/express @types/node` <br />

#### Initalisation de tslint
`yarn tslint --init`

#### Webpack config
`yarn add --dev webpack-cli webpack ts-loader`
